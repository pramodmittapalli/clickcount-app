About

  • Clickcount
  • A cross-platform mobile app built with web standards
  • Based on jQTouch and PhoneGap
  • The counter at the top shows the number of times that you have pressed the "Click me!" button. Each time you press it, the number will increase by one.
  • You can set the counter back to zero again by pressing the "Reset" button in the toolbar.
  • This is not designed to be a functional app - it is a proof of concept to show how easy it is to make a simple app with web technologies.
